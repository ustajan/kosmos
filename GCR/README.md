# Galactic Cosmic Rays (GCR)

In this sub-directory we model the galactic cosmic rays, based on the existing parametrized models for the GCR flux. 

To produce the plots of proton spectra and flux vs latitude run `gcr.py`. 

The `ez3d` directory contains a grasshopper model for determining the EZ3D signal from a single 20 MeV electron pulse against a DU target.
